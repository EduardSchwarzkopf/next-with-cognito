A demo Next.js application (using experimental app folder) with AWS Cognito and NextAuth.js

Add your cognito variables into `.env.local`

```javascript
COGNITO_CLIENT_ID = 
COGNITO_CLIENT_SECRET = 
COGNITO_ISSUER = 
```

Install all packages

```bash
npm run install
```

Run the development server:

```bash
npm run dev
```

Enjoy!
